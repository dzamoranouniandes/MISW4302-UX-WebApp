## Tecnologías del proyecto

* Framework de UI: Angular version 21
* Sistema de Diseño: Bootstrap

## Estructura del proyecto

Ruta base del código de la app `src`:
* `app/<component-name>/<component-name>.component`: Librería de componentes y pantallas de la aplicación.
* `app/app.component`: Componente principal de la aplicación que configura el layout y la navegación.

## Ejecución de aplicación

Descargar el proyecto y ejecutar los siguientes comandos sobre la carpeta raíz del proyecto.

```shell
npm install
npm run start
```

Abrir la web app en el navegador en la URL indicada en la consola (ej: http://localhost:4200/).
